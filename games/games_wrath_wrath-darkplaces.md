# Compilation
```shell
$ mkdir -p ~/src
$ cd src
$ git clone https://github.com/KillPixelGames/wrath-darkplaces.git
$ cd wrath-darkplaces
# pacman -S libpng freetype2 libcurl-compat libcurl-gnutls sdl2
$ make sdl2-release

```

# Usage
```shell
$ ~/src/wrath-darkplaces/darkplaces-sdl -wrath -width 1920 -height 1080 -fullscreen -basedir ~/games/wrath
```
