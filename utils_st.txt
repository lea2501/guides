# Installation
```
$ mkdir -p ~/src
$ cd ~/src
$ git clone https://git.suckless.org/st
$ cd st
$ vim config.def.h
# make clean install
```

# Usage
```
$ st -f "Dejavu sans mono:pixelsize=24:antialias=true:autohint=false" -g 80x24
$ st -f "Dejavu sans mono:pixelsize=36:antialias=true:autohint=false" -g 80x24
$ st -f "Bitstream vera sans mono:pixelsize=24:antialias=true:autohint=false" -g 80x24
```
