
# Play DVDs via vlc
```shell
# apt-get install libdvd-pkg
# dpkg-reconfigure libdvd-pkg
```

# Play Blu-Ray via vlc
```shell
$ mkdir ~/.config/aacs
$ cd ~/.config/aacs
$ wget http://vlc-bluray.whoknowsmy.name/files/KEYDB.cfg
```
